# Rock–Paper–Scissors (Dart)

A simple command-line Rock–Paper–Scissors game built to practice Dart basics like enums, input handling, and control flow.

## How to Play

* Enter:

    * `r` = Rock
    * `p` = Paper
    * `s` = Scissor
* Computer picks a random move.
* Result is printed: **Player Won**, **Comp Won**, or **Draw**.
* Type `q` or press **Ctrl + D** to exit.

## Features

* Enum-based move handling
* Randomized computer choice
* Continuous gameplay loop
* Input validation

## Run the Game

```bash
dart run main.dart
```

## Game Rules

* Rock > Scissor
* Scissor > Paper
* Paper > Rock
